# D&D DM Sidekick API

 The backend API to the 'DM Sidekick' tool
 

# Requirements
- Python 3.9
- PyEnv

# Installation
```bash
git clone git@github.com:stefanjcollier/DnD_DM_Sidekick_API.git
cd DnD_DM_Sidekick_API
pyenv install
```

# Run
```bash
cd DnD_DM_Sidekick_API
pyenv shell
python manage.py
```
